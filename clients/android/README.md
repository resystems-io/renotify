# Renotify Android Client

Kotlin-based Android application for receiving notifications,
capturing human decisions, and sending interjection commands.

## Prerequisites

- Android SDK (API level 26+)
- Gradle (wrapper included after M-01 scaffolding)

## Build

```bash
make          # assembleRelease
make test     # run tests
make clean    # remove build artifacts
```

See the implementation plan item **M-01** in
[renotify-refinements.md](../../docs/renotify-refinements.md) for
scaffolding details.
